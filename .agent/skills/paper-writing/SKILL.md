# Scientific Paper Skill

## Goal
Help draft and refine the final paper while preserving academic accuracy and compliance with project requirements.

## Instructions
- Always follow the selected journal template and section structure.
- Never invent results, metrics, citations, experiments, or claims.
- Only write from evidence found in:
  - the repository
  - experiment logs
  - provided literature review
  - explicitly supplied notes/results
- Keep the abstract under 300 words.
- Maintain a formal academic tone.
- Flag any sentence that sounds unsupported.
- Prefer concise and specific writing over generic wording.
- Reduce text reuse from the literature review; paraphrase properly and cite sources.
- Keep an "Author contributions" section at the end.
- When drafting results, use actual values only.
- When unsure, insert a clear placeholder like [NEED RESULT] instead of guessing.

## Constraints
- Do not fabricate references.
- Do not copy long passages from source PDFs.
- Do not claim experiments were run unless result files/logs support it.
